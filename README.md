# Expense Tracker (Java)

A simple console-based Expense Tracker developed in Java to record, view, and manage daily expenses.

* **Student Name:** Kriti

* **Course:** BYOP (Bring Your Own Project)
* **Language Used:** Java

## Overview

This project is a basic Expense Tracker application that allows users to:

* Add expenses with category and amount
* View all recorded expenses
* Calculate total spending

It helps in understanding how to manage financial data using core Java concepts like classes, objects, ArrayList, and user input handling.

## Features

* Add new expenses with category and amount
* View list of all expenses
* Calculate total spending
* Menu-driven interface
* Uses ArrayList for dynamic data storage

## Tech Stack

* **Programming Language:** Java
* **Concepts Used:**

  * Classes and Objects
  * ArrayList
  * Loops and Conditionals
  * User Input (Scanner)

## How to Run

### 1. Clone the repository

```bash
git clone https://github.com/codeWkrits/Expense-Tracker-Java.git
```

### 2. Navigate to the folder

```bash
cd expense-tracker
```

### 3. Compile the program

```bash
javac Main.java
```

### 4. Run the program

```bash
java Main
```

## Usage

After running the program, you will see a menu:

```
--- EXPENSE TRACKER ---
1. Add Expense
2. View Expenses
3. View Total Spending
4. Exit
```

### Example Flow:

* Choose 1 → Add category and amount
* Choose 2 → View all expenses
* Choose 3 → See total spending

## Limitations

* Data is not stored permanently (resets after program ends)
* No graphical interface (CLI-based only)
* No editing or deleting expenses

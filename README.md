# Expense Tracker (Java)

## Overview
This is a simple console-based Expense Tracker developed in Java. It allows users to manage their daily expenses efficiently.

## Features
- Add expenses with category and amount
- View all expenses
- Calculate total spending

## Technologies Used
- Java
- ArrayList
- Scanner

## How to Run
1. Compile the program:
   javac ExpenseTracker.java
2. Run the program:
   java ExpenseTracker

## Project Structure
- Expense class: stores category and amount
- Main class: handles menu and operations
